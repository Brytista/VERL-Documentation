# VERL 0.7 Documentation Snapshot

This directory contains a locally organized snapshot of the VERL 0.7 documentation. The source material was collected from the official VERL documentation site and saved as text files for easier local browsing, search, and use by AI agents.

The filename prefixes (for example, `a1`, `b2`, and `j14`) are a local filing convention used to group related topics and keep the files in a stable order. They are not part of the original upstream documentation.

This collection is intended to help users and AI agents quickly find relevant VERL references when studying, implementing, or debugging VERL-based workflows in this repository.

## Classification

1. `a` series: Quickstart
2. `b` series: Programming guides
3. `c` series: Data preparation
4. `d` series: Configuration
5. `e` series: PPO examples
6. `f` series: Algorithms
7. `g` series: PPO trainer and worker components
8. `h` series: Performance tuning guide
9. `i` series: Adding new models
10. `j` series: Advanced features
11. `k` series: Hardware support
12. `l` series: API references
13. `m` series: Blog posts, FAQ, and development notes

## Additional Documentation
Additional documentation (copied from VERL github repository) is present in the 'docs' folder. There are significant overlap between the documentation from github repository and VERL official documentation website.
